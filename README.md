# 4-bit---synchronous---up-counter
4-bit synchronous up counter in Verilog HDL with complete design, testbench, and simulation results.

# Overview
This is my first project, to implements a 4-bit synchronous up counter using Verilog HDL.

# Features
- 4-bit counter
- Positive edge triggered
- Active-high reset
- Testbench and simulation included

## Files
- counter.v
- counter_tb.v

## Tools Used
- Verilog HDL
- Virtual box Verilator
- GTKWave for waveform simulation
